# 🌸 Wellness Tracker

A beautiful cherry blossom themed personal wellness dashboard built with HTML, CSS & JavaScript.

## Features
- 🍱 Indian vegetarian diet plan with daily meal tracking
- 💪 5-day machine-based gym plan 
- ✨ Wellness routines — vitamins, skincare AM/PM, hair care
- 📋 Dynamic task manager — add, complete & delete tasks
- 🌸 Daily goal % with streak & progress calendar
- 🎊 Confetti animation on task completion
- 💾 Auto-saves daily progress to local storage
- 🌸 Falling cherry blossom petals animation

## Built with
HTML · CSS · Vanilla JavaScript
